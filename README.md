# yomigaeru launcher

## overview

classic PC-game launcher for HSP3.

## how to use

次の書籍に付属するCD-ROMをあらかじめインストールしてから、実行します。

「みんながコレで燃えた! NEC8ビットパソコン PC-8001・PC-6001 永久保存版」

「蘇るPC-8801伝説 永久保存版」

「蘇るPC-9801伝説 永久保存版」

「蘇るPC-9801伝説 永久保存版 第2弾」

リストからソフトを選択して、「launch」ボタンを押すと、該当するソフトを起動します。
